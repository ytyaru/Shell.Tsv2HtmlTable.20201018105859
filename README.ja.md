# Tsv2HtmlTable

　TSVをHTMLのtableタグに変換する。

# デモ

* [demo](https://ytyaru.github.io/Shell.Tsv2HtmlTable.20201018105859/)

# 特徴

　以下のようなtableタグを作成できる。

* 1行目がヘッダである
* 1列目がヘッダである
* 1行目と1列目がヘッダである
