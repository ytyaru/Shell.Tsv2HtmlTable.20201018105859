TSV to HTML table examples

1st row Header

echo -e "Id\tName\n0\tA\n1\tB" | ./th_row.sh
<table><tr><th>Id</th><th>Name</th></tr><tr><td>0</td><td>A</td></tr><tr><td>1</td><td>B</td></tr></table>
IdName
0A
1B

1st column Header

echo -e "Id\t0\t1\nName\tA\tB" | ./th_column.sh
<table><tr><th>Id</th><td>0</td><td>1</td></tr><tr><th>Name</th><td>A</td><td>B</td></tr></table>
Id01
NameAB

1st row & column Header

echo -e "\tA\tB\n1\tX\tY\n2\ta\tb" | ./th_row_column.sh
<table><tr><th></th><th>A</th><th>B</th></tr><tr><th>1</th><td>X</td><td>Y</td></tr><tr><th>2</th><td>a</td><td>b</td></tr></table>
AB
1XY
2ab
